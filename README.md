# SpaceX Falcon 9 Landing Predictions

IBM Data Science capstone project predicting whether the Falcon 9 first stage will land successfully.

## Project Goal

SpaceX can reduce launch costs by reusing the Falcon 9 first stage. This project estimates the likelihood of a successful first-stage landing using public launch data, exploratory analysis, dashboarding, and machine learning.

## Files

- `jupyter-labs-spacex-data-collection-api.ipynb` - API data collection.
- `jupyter-labs-webscraping.ipynb` - web scraping workflow.
- `labs-jupyter-spacex-Data wrangling.ipynb` - cleaning and feature preparation.
- `jupyter-labs-eda-sql-coursera_sqllite.ipynb` - SQL-based exploratory analysis.
- `edadataviz.ipynb` - exploratory visualizations.
- `lab_jupyter_launch_site_location.ipynb` - launch-site location analysis.
- `SpaceX_Machine Learning Prediction_Part_5.ipynb` - model training and prediction.
- `spacex_dash_app.py` - Plotly Dash app.
- `Screenshot_dashboard.png` - dashboard screenshot.
- `ds-capstone-ibm-artyom-ashigov.pdf` - final capstone report.

## Workflow

1. Collect launch data from APIs and web sources.
2. Clean and combine launch records.
3. Explore patterns by launch site, payload, orbit, and landing outcome.
4. Build visualizations and an interactive dashboard.
5. Train classification models to predict landing success.

## Tools

Python, pandas, SQL, scikit-learn, Plotly Dash, Folium, and Jupyter Notebook.
